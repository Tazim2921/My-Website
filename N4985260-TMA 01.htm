what is XML and how is it used?

XML (eXtensible Markup Language) is a markup language designed to both store and transport data. Where HTML would focus on data displayed, XML is more about what the data is. With using a custom set of tags defined by the user, it enables it to be readable to humans and machines.

an example of XML could look like this:

<book>

<title>XML for Beginners</title>

<author>Jane Doe</author>

</book>

This format enables information to be easily shared between systems or applications. XML is mostly used for RSS feeds, web services and configuration files as it allows different systems to share structured information. Also, as it uses a hierarchical structure along with its simplicity, is what makes it ideal for data interchange across different platforms.
